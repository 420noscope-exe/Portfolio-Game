- Statemachine NavMesh AI
- Procedural Generation System
- Fast Paced, Physics RigidBody Based PlayerController
- Dash and Stamina System
- 5 Different Weapons
- Pickup Items
- All Assets are Made with Royalty Free or Open Source Tools
